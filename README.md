# Amarel Interface
A web interface for the Rutgers supercomputer Amarel 

## Usage
1. Clone the repository
```sh
git clone https://github.com/jeevanshah07/amarel-interface.git && cd amarel-interface
```
2. Install the required packages
```sh
pipenv install
```
3. Start the local webserver
```sh
pipenv run flask run --debug
```
4. You can now visit `localhost:5000` to use the webserver! 
**Note that this project is still in development and errors are bound to occur**
